# Aufgabe: Bewerbungsformular mit Datenspeicherung

Funktionsweise:
Der Nutzer soll ein Bewerbungsformular ausfüllen können und durch einen Buttonklick die eingegebenen Daten 
(Name, Alter, etc.) in einer JSON-Datei abgespeichert werden. Eine Auswahl von Dokumenten soll ebenfalls
möglich sein, welche dann ebenfalls gespeichert werden sollen.
War der Upload erfolgreich oder ging etwas schief, so soll eine entsprechende Benachrichtigung erscheinen.


Der Backend-Code soll in Python geschrieben werden und durch einen Compiler nach Wahl in Web Assembly (WASM)
compiled werden.
Eine README soll die Verwendung und hierfür nötige Schritte wie z.B. das Installieren von Plugins o.Ä.
beschreiben.
